# Branch base per avviare un nuovo progetto Streamlit

## Virtual env
Si suggerisce di creare un virtual environment (e.g., venv, pipenv).
```pip install pipenv```

## Run

Avviare il virtual env:

```pipenv shell```

Installare le dependencies:

```pip install -r requirements.txt```
